
# 📊 Customer Product & Profitability Analysis Dashboard

This project is built using **Python, Pandas, Matplotlib, and Streamlit** to analyze customer, product, sales, and profitability data through interactive visualizations.

## Features
- Data Cleaning
- Sales Analysis
- Profit Analysis
- Customer Analysis
- Product Analysis
- Market-wise Profit Analysis
- Region-wise Sales Analysis
- Interactive Dashboard

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit


## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

The cleaned dataset is not included because it exceeds GitHub's file size limit.
